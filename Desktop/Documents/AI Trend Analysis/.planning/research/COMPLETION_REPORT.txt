VALIDATION APPROACH (PHASE 0 RESEARCH)
================================================================================

Before committing full development resources, conduct 2-4 week Phase 0 validation:

VALIDATION 1: Core Signal Feasibility (1 week)
- Extract signals from 5 sample companies
- Validate computation time and API access
- Gate: All signals extractable in <5 min per company

VALIDATION 2: Signal Correlation (1 week)
- Compute signals for 20 companies (10 credible AI, 10 questionable)
- Verify signal independence and outcome correlation
- Gate: Correlation with fraud outcome >0.3 for all signals

VALIDATION 3: Ensemble Accuracy (1 week)
- Simple ensemble on 20 companies
- Calculate accuracy, false positive rate, false negative rate
- Gate: Accuracy 65-75%, false positive <20%, false negative <25%

VALIDATION 4: Data API Stability (1 week)
- Test SEC Edgar, GitHub, USPTO, Seeking Alpha APIs
- Verify rate limits and fallback mechanisms
- Gate: ~95% success rate, rate limiting understood

Success criterion: Phase 0 backtest accuracy >65% on known cases → proceed to Phase 1
